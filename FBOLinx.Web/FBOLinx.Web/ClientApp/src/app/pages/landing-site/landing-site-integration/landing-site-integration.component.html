Integration

FBOlinx integrates with The Paragon Network. The Paragon Network is an exclusive membership of independent FBOs that provide an elite experience to general aviation travelers in North America. Each FBO joining The Paragon Network goes through a comprehensive audit of facility and services offered to ensure the quality of the member base. A set of core standards has been developed to which each member FBO must adhere and maintain in order to remain in the group.
